‚úÖ What is a Function?

A function is a named block of code that performs a specific task and can be called multiple times throughout a program. Functions help organize code, avoid repetition, and improve readability.

üß† Syntax

returnType functionName(parameterList) {
    // code to execute
    return value; // if returnType is not void
}

Example:

int add(int a, int b) {
    return a + b;
}

üìå Calling a Function

int result = add(5, 3); // Calls add() with 5 and 3

Functions with No Return Value:

void printMessage() {
    std::cout << "Hello, World!" << std::endl;
}

Call:

printMessage();

‚ö†Ô∏è Important Rules

  1. Function must be declared before it is called (or use a prototype).
  2. Parameters are optional; a function can have no parameters.
  3. Return type can be void if the function does not return a value.
  4. Functions can be overloaded (same name, different parameters).

üßæ Example Program

#include <iostream>
using namespace std;

// Function prototype
int multiply(int x, int y);

int main() {
    int a = 4, b = 5;
    cout << "Product: " << multiply(a, b) << endl;
    return 0;
}

// Function definition
int multiply(int x, int y) {
    return x * y;
}

üîÑ Functions with Parameters and Loops

#include <iostream>
using namespace std;

void printNumbers(int n) {
    for (int i = 1; i <= n; i++) {
        cout << i << " ";
    }
    cout << endl;
}

int main() {
    printNumbers(5);  // Output: 1 2 3 4 5
    return 0;
}
