🔢 The sgn Function

✅ What is sgn?

The sign function (sgn) is a utility function that determines the sign of a number.

  • Returns +1 if the number is positive.
  • Returns 0 if the number is zero.
  • Returns -1 if the number is negative.
  • Often used in mathematical computations, comparisons, sorting algorithms, and physics simulations.

Note: C++ does not have a built-in sgn function, but it is trivial to implement.

📌 How sgn Works

Logic of sgn(x):

  1. If x > 0 → return 1
  2. Else if x < 0 → return -1
  3. Else → return 0

Mathematically:

$$ sgn(x) = \begin{cases} 1 & \text{if } x > 0 \ 0 & \text{if } x = 0 \ -1 & \text{if } x < 0 \end{cases} $$

🧾 Example – Implementing sgn in C++ 

#include <iostream>
using namespace std;

// Generic template for sgn
template <typename T>
int sgn(T value) {
    return (T(0) < value) - (value < T(0));
}

int main() {
    int a = 10;
    int b = -5;
    int c = 0;

    cout << "sgn(" << a << ") = " << sgn(a) << endl; // 1
    cout << "sgn(" << b << ") = " << sgn(b) << endl; // -1
    cout << "sgn(" << c << ") = " << sgn(c) << endl; // 0

    double d = -3.14;
    cout << "sgn(" << d << ") = " << sgn(d) << endl; // -1

    return 0;
}

Output:

sgn(10) = 1
sgn(-5) = -1
sgn(0) = 0
sgn(-3.14) = -1

🔄 How It Works

  • (T(0) < value) evaluates to 1 if value > 0, 0 otherwise.
  • (value < T(0)) evaluates to 1 if value < 0, 0 otherwise.
  • Subtracting gives +1, 0, or -1, matching the sign of the number.
  • Works for integers, floating-point numbers, or any comparable numeric type.

⚠️ Important Rules

  1. C++ standard does not provide sgn, so you must implement it.
  2. Works for any numeric type, including int, float, double, long long.
  3. Useful in mathematical expressions, sorting, and comparisons.
  4. Can also be implemented using simple if-else statements if preferred: 
int sgn(int x) {
    if (x > 0) return 1;
    else if (x < 0) return -1;
    else return 0;
}
