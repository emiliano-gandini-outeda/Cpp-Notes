Loops are control structures that allow a program to execute a block of code multiple times, either for a set number of iterations or until a certain condition is no longer met.

üîç How It Works

The for loop is best suited when:

  • You know exactly how many times you want to repeat something.
  • The number of iterations can be determined before entering the loop.

It contains three parts in the header:

  1. Initialization ‚Üí Executed once before the loop starts.
  2. Condition ‚Üí Checked before each iteration. If false, loop ends.
  3. Update ‚Üí Executed after each iteration (increment, decrement, etc.).

The for loop is compact because all these elements are declared in one line.

‚úÖ Example

#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 10; i++) {
        cout << "Square of " << i << " is " << i*i << endl;
    }
    return 0;
}

How it runs:

  1. i starts at 1.
  2. Condition i <= 10 is checked.
  3. Body runs.
  4. i++ increments i.
  5. Condition is checked again, and so on.

üìå Use Cases

  • Iterating through arrays or vectors when size is known.
  • Running an operation a fixed number of times.
  • Generating a table of values (e.g., multiplication tables).
  • Counting up or down in a predictable sequence.

2Ô∏è‚É£ while Loop

üîç How It Works

The while loop is used when:

  • You don‚Äôt know the exact number of iterations in advance.
  • You want the loop to run as long as a condition remains true.

The condition is checked before each iteration, meaning the body may never run if the condition is false from the start.

‚úÖ Example

#include <iostream>
using namespace std;

int main() {
    int number;
    cout << "Enter numbers (0 to stop): ";
    cin >> number;

    while (number != 0) {
        cout << "You entered: " << number << endl;
        cin >> number;
    }

    cout << "Loop ended." << endl;
    return 0;
}

How it runs:

  1. Condition is checked first.
  2. If true, body executes.
  3. After body, condition is checked again.
  4. Loop repeats until condition becomes false.

üìå Use Cases

  • Reading user input until a sentinel value is entered.
  • Waiting for a certain event or condition (e.g., data availability).
  • Processing items from a queue until empty.
  • Running a task until a success flag is set.

3Ô∏è‚É£ do...while Loop

üîç How It Works

  • Similar to while, but the condition is checked after the loop body.
  • Guarantees that the loop body executes at least once, even if the condition is initially false.

‚úÖ Example

#include <iostream>
using namespace std;

int main() {
    int number;
    do {
        cout << "Enter a positive number (0 to exit): ";
        cin >> number;
        if (number > 0) {
            cout << "You entered: " << number << endl;
        }
    } while (number != 0);

    cout << "Loop ended." << endl;
    return 0;
}

How it runs:

  1. Body executes once without checking condition.
  2. Condition is checked.
  3. If true, loop repeats; if false, it ends.

üìå Use Cases

  • Menus that should be shown at least once.
  • Input validation where you want the user to be prompted at least once.
  • Repetitive tasks that must happen before knowing if continuation is needed.

üö´ Loop Control Statements

  • break; ‚Üí Immediately exits the loop.
  • continue; ‚Üí Skips the rest of the current iteration, jumps to the next one.
