üìå What is scanf?

scanf is a C standard library function defined in <stdio.h> used to read input from the user via the keyboard. It stores the input into the memory addresses of variables.

üß† Syntax:

scanf("format string", &var1, &var2, ...); // Read input into variables
  • The format string tells scanf what kind of data to expect (%d, %f, %c, etc.).
  • The & operator (address-of) is used to pass the memory address where the input will be stored.

‚úÖ Example:

#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age); // Reads an integer and stores it in age
    printf("You entered: %d\n", age);
    return 0;
}

‚ö†Ô∏è Common Mistakes:

  • Forgetting the & before variable names.
  • Using the wrong format specifier for the variable type.
  • Not handling leftover input (e.g., newline characters when reading characters or strings).

‚úÖ Multiple Inputs

int c1, c2, c3, c4, c5;
scanf("%d %d %d %d %d", &c1, &c2, &c3, &c4, &c5);
  • User inputs five integers separated by spaces.
  • Each %d matches an integer input and stores it in the corresponding variable.
