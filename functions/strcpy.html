‚úÖ What is strcpy?

strcpy is a standard library function used to copy a string from one character array to another. The destination string must have enough space to hold the source string including the null character \0.

üß† Syntax

char *strcpy(char *destination, const char *source);
  • destination: The array where the string will be copied.
  • source: The string to copy.
  • Returns a pointer to destination.

üìå Important Rules

  1. Destination must be large enough to store the source string and the null character.
  2. Overlapping strings in source and destination cause undefined behavior.
  3. Strings must be null-terminated (\0).
  4. Defined in <string.h>.

üßæ Example Program

#include <stdio.h>
#include <string.h>

int main() {
    char src[] = "Hello, World!";
    char dest[50];  // Ensure enough space

    strcpy(dest, src);

    printf("Source: %s\n", src);
    printf("Destination: %s\n", dest);

    return 0;
}

üîÑ Copying Multiple Strings

#include <stdio.h>
#include <string.h>

int main() {
    char name1[] = "Emiliano";
    char name2[20];

    strcpy(name2, name1);   // Copy name1 into name2

    printf("name2: %s\n", name2);

    return 0;
}

‚ö†Ô∏è Important Notes

  1. Always ensure destination buffer is large enough.
  2. Use strncpy for safer copying with maximum length.
  3. strcpy does not check bounds, so careless use may cause buffer overflow.
