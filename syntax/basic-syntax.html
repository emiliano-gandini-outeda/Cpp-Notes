Basic Syntax Rules of C++

1. Structure of a C Program

A minimal C program looks like this:

#include <stdio.h>  // Preprocessor directive for input/output

int main() {
    // Your code here
    return 0;       // Return statement indicates successful program termination
}
  • #include <stdio.h> includes the standard input/output library.
  • int main() is the main function where execution starts.
  • return 0; indicates the program ended successfully.

2. Comments

C supports two types of comments:

// Single-line comment

/*
 Multi-line comment
*/

3. Case Sensitivity

  • C is case-sensitive.
  • variable, Variable, and VARIABLE are all different identifiers.

4. Semicolons ;

  • Each statement must end with a semicolon.
int x = 5;    // ✅ Correct
int y = 10    // ❌ Error: missing semicolon

5. Variables and Data Types

Variables must be declared with a type before use.

Type Size (typical) Description Example
int 4 bytes Whole numbers int age = 30;
float 4 bytes Decimal numbers (single precision) float pi = 3.14f;
double 8 bytes Decimal numbers (double precision) double e = 2.718;
char 1 byte Single character char grade = 'A';
_Bool 1 byte Boolean (0 or 1) from <stdbool.h> _Bool flag = 1;

6. Variable Declaration and Initialization

int x;             // Declaration (uninitialized)
int y = 10;        // Declaration + initialization
float pi = 3.14;
char letter = 'A';

7. Naming Rules for Variables

  • Must start with a letter or underscore (_)
  • May contain letters, digits, and underscores
  • Cannot use C reserved keywords (e.g., int, return, if)
  • Are case-sensitive

✅ Valid:

int count;
float _value;
char grade1;

❌ Invalid:

int 1stValue;  // Starts with digit
int return;      // Reserved keyword

8. Output and Input

In C, input/output is done using functions from the <stdio.h> library:

  • printf() is used for output
  • scanf() is used for input

Example:

#include <stdio.h>

int main() {
    int age;

    printf("Enter your age: ");  // Output message
    scanf("%d", &age);           // Input from user

    printf("You are %d years old.\n", age);  // Output result

    return 0;
}

Notes:

  • %d is a format specifier for integers.
  • The & (address-of operator) is required with scanf.

9. Basic Arithmetic Operators

Operator Operation
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulo (remainder)

Example with integer vs floating-point division:

int a = 7, b = 2;
int intDiv = a / b;      // Result: 3 (integer division)

float x = 7.0f, y = 2.0f;
float floatDiv = x / y;  // Result: 3.5 (floating-point division)

Other arithmetic examples:

int a = 10, b = 3;
int sum = a + b;     // 13
int diff = a - b;    // 7
int prod = a * b;    // 30
int div = a / b;     // 3
int mod = a % b;     // 1

10. Constants

Use const to declare constants (read-only variables):

const double PI = 3.14159;

11. Example Program with Variables and Types

#include <stdio.h>
#include <stdbool.h>

int main() {
    int age = 25;
    float height = 1.75f;
    char grade = 'B';
    _Bool isStudent = 1;  // Use <stdbool.h> if you want true/false

    printf("Age: %d\n", age);
    printf("Height: %.2f meters\n", height);
    printf("Grade: %c\n", grade);
    printf("Student? %s\n", isStudent ? "Yes" : "No");

    return 0;
}
