📦 Hash Maps

✅ What is a Hash Map?

A hash map (also called a hash table) is a data structure that stores key-value pairs. It provides fast insertion, deletion, and lookup, typically in O(1) average time.

  • Keys are unique identifiers.
  • Values are associated data for each key.
  • Internally, a hash function converts a key into an index in an array (bucket).

Hash maps are widely used for dictionaries, counting frequencies, caching, and more.

🧠 Core Concepts

Term Description
Key Unique identifier used to access a value
Value Data associated with a key
Hash Function Algorithm converting a key into an array index
Bucket / Slot Position in the array to store key-value pairs
Collision Occurs when two keys hash to the same index
Load Factor Ratio of elements to number of buckets; affects resizing
Rehashing Increasing the number of buckets when load factor exceeds a threshold

📌 Collision Handling Techniques

  1. Chaining
    • Each bucket stores a linked list of key-value pairs.
    • Colliding elements are added to the list.

    Example:

    Bucket 2 → ("Alice", 25) → ("Bob", 30) → NULL
  2. Open Addressing
    • Store colliding elements in another empty slot.
    • Techniques:
      • Linear Probing: Move sequentially to the next free slot.
      • Quadratic Probing: Move by increasing quadratic steps.
      • Double Hashing: Use a second hash function to find the next slot.

🧾 Example Program (STL unordered_map)

#include <iostream>
#include <unordered_map>
using namespace std;

int main() {
    // Create a hash map
    unordered_map<string, int> ages;

    // Insert key-value pairs
    ages["Alice"] = 25;
    ages["Bob"] = 30;
    ages["Charlie"] = 22;

    // Access value by key
    cout << "Alice's age: " << ages["Alice"] << endl;

    // Update a value
    ages["Alice"] = 26;
    cout << "Alice's new age: " << ages["Alice"] << endl;

    // Check if key exists
    if(ages.find("David") == ages.end())
        cout << "David not found\n";

    // Iterate over hash map
    cout << "\nAll key-value pairs:\n";
    for(auto &pair : ages) {
        cout << pair.first << ": " << pair.second << endl;
    }

    // Remove a key
    ages.erase("Bob");

    // Size of hash map
    cout << "\nNumber of elements: " << ages.size() << endl;

    return 0;
}

🔄 Common Operations

Operation STL Method / Description

Access map[key] or map.at(key) (throws exception if key not found)

📌 Advanced Examples

Counting Frequency of Elements

#include <iostream>
#include <unordered_map>
using namespace std;

int main() {
    int arr[] = {1, 2, 2, 3, 3, 3};
    unordered_map<int, int> freq;

    for(int num : arr) {
        freq[num]++;  // Increment count for each element
    }

    for(auto &p : freq) {
        cout << "Number " << p.first << " appears " << p.second << " times\n";
    }
}

Using Custom Keys (Struct)

#include <iostream>
#include <unordered_map>
using namespace std;

struct Point {
    int x, y;
    bool operator==(const Point &other) const {
        return x == other.x && y == other.y;
    }
};

// Custom hash function
struct PointHash {
    size_t operator()(const Point &p) const {
        return hash<int>()(p.x) ^ hash<int>()(p.y << 1);
    }
};

int main() {
    unordered_map<Point, string, PointHash> points;
    points[{1,2}] = "A";
    points[{3,4}] = "B";

    for(auto &p : points) {
        cout << "(" << p.first.x << "," << p.first.y << ") -> " << p.second << endl;
    }
}

⚠️ Important Rules

  1. Keys must be unique, values can repeat.
  2. Use a good hash function to minimize collisions.
  3. unordered_map provides O(1) average time, worst-case O(n).
  4. map (ordered map) stores keys sorted, with O(log n) operations.
  5. STL unordered_map automatically rehashes when load factor exceeds threshold.
