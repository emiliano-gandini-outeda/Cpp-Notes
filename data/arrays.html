âœ… What is an Array?

An array is a fixed-size collection of elements of the same type, stored contiguously in memory. Arrays allow direct access to any element in constant time O(1) using an integer index. However, their size is fixed at declaration, so it cannot grow or shrink during runtime.

ðŸ§  Syntax

type arrayName[size];

Examples:

int numbers[5];               // Declares an array of 5 integers
int values[5] = {1, 2, 3, 4, 5}; // Full initialization
int partial[5] = {10, 20};       // Remaining elements set to 0
int inferred[] = {1, 2, 3};      // Size inferred as 3

ðŸ”„ Accessing and Modifying Elements

int x = values[0];   // Access first element
values[1] = 100;     // Modify second element

Important rules:

  1. Index starts at 0.
  2. Accessing outside bounds â†’ undefined behavior.
  3. All elements must be of the same type.
  4. Size must be constant (at compile time in traditional C++).

Arrays are efficient for memory-contiguous storage and constant-time access, but their fixed size makes them less flexible than dynamic structures. They are ideal when data size is known and frequent direct access is required.
