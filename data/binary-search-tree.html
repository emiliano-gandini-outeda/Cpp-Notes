‚úÖ What is a BST?

A Binary Search Tree (BST) is a special type of binary tree with an ordering property:

  • Values in the left subtree are smaller than the node.
  • Values in the right subtree are greater than the node.

This property allows efficient searching, insertion, and deletion.

  • Average complexity: O(log n) (balanced tree).
  • Worst case: O(n) (degenerate/unbalanced tree).

üß† Syntax (Node Definition)

struct Node {
    int data;
    Node* left;
    Node* right;
};

üßæ Example Program ‚Äì Insert & Search in BST

#include <iostream>
using namespace std;

struct Node {
    int data;
    Node* left;
    Node* right;
};

Node* createNode(int value) {
    return new Node{value, nullptr, nullptr};
}

Node* insert(Node* root, int value) {
    if (!root) return createNode(value);
    if (value < root->data)
        root->left = insert(root->left, value);
    else
        root->right = insert(root->right, value);
    return root;
}

bool search(Node* root, int value) {
    if (!root) return false;
    if (root->data == value) return true;
    if (value < root->data) return search(root->left, value);
    return search(root->right, value);
}

int main() {
    Node* root = nullptr;
    root = insert(root, 10);
    root = insert(root, 5);
    root = insert(root, 15);
    root = insert(root, 7);

    cout << "Search 7: " << (search(root, 7) ? "Found" : "Not Found") << endl;
    cout << "Search 12: " << (search(root, 12) ? "Found" : "Not Found") << endl;

    return 0;
}

‚ö†Ô∏è Important Notes

  1. BST property ensures fast search compared to unsorted arrays/lists.
  2. Operations: search, insert, delete all rely on ordering.
  3. Works best when balanced (height ‚âà log n).
  4. Can degrade to a linked list if values are inserted in sorted order.
