‚úÖ What is a Pointer?

A pointer is a variable that stores the memory address of another variable. Instead of holding a value directly, it points to where the value is stored in memory.

üß† Syntax

type *pointerName;
  • type: The data type of the variable the pointer will point to.
  • *: Declares a pointer.

üìå Basic Operations

Operation Symbol Meaning
Address-of & Gets the address of a variable
Dereference * Accesses the value at a memory address

üßæ Example Program 

#include <stdio.h>

int main() {
    int x = 10;
    int *ptr = &x;  // Pointer to x

    printf("Value of x: %d\n", x);
    printf("Address of x: %p\n", &x);
    printf("Pointer value (address): %p\n", ptr);
    printf("Value pointed to by ptr: %d\n", *ptr);

    return 0;
}

üîÑ Changing Values via Pointers 

#include <stdio.h>

int main() {
    int x = 5;
    int *p = &x;

    *p = 20; // Changes x through pointer

    printf("x = %d\n", x);

    return 0;
}
