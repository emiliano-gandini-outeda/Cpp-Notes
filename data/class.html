‚úÖ What is a class?

A class is a user-defined data type that allows you to combine data (attributes) and functions (methods) into a single unit. Classes are the foundation of object-oriented programming in C++.

üß† Syntax

class ClassName {
public:
    type member1;
    type member2;

    void method1();
    returnType method2(type param);
};
  • ClassName: The name of the class.
  • members: Variables or functions that belong to the class.
  • public, private, protected control access to members.

üìå Declaring Objects

ClassName obj1, obj2;
  • Objects are instances of a class.
  • Access public members using the dot operator ..

üßæ Example Program

#include <iostream>
using namespace std;

class Person {
public:
    string name;
    int age;

    void greet() {
        cout << "Hello, my name is " << name << " and I am " << age << " years old.\n";
    }
};

int main() {
    Person p1;
    p1.name = "Emiliano";
    p1.age = 30;

    p1.greet();

    return 0;
}

üîÑ Constructor Example

#include <iostream>
using namespace std;

class Point {
public:
    int x, y;

    // Constructor
    Point(int px, int py) {
        x = px;
        y = py;
    }

    void display() {
        cout << "(" << x << ", " << y << ")\n";
    }
};

int main() {
    Point p(5, 10);
    p.display();

    return 0;
}
