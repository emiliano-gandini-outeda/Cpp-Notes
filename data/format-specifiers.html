‚úÖ What are Format Specifiers?

Format specifiers are placeholders in functions like printf() and scanf() that tell the compiler the type, size, and format of the data to display or read. They start with % followed by characters that define formatting.

üß† Syntax

printf("format_string", values...);
scanf("format_string", &variables...);

Example:

printf("Value: %d\n", 42);
scanf("%f", &number);

üîπ Modifiers for Width & Precision

  • Width ‚Üí minimum field size (%6d).
  • Precision ‚Üí decimal or string length control (%.2f).
  • Zero-padding ‚Üí %06d.

Example:

printf("%6d", 42);       // "    42"
printf("%.2f", 3.14159); // "3.14"
printf("%06d", 42);      // "000042"

‚ö†Ô∏è Important Rules

  1. Match type with specifier.
  2. scanf() needs & for variables (except arrays/strings).
  3. %s stops at whitespace unless width is specified.

üßæ Example Program

#include <stdio.h>

int main() {
    int age = 25;
    unsigned int score = 5000;
    double pi = 3.1415926535;
    char grade = 'A';
    char name[] = "Alice";

    printf("Name: %s\n", name);
    printf("Age: %d\n", age);
    printf("Score: %u\n", score);
    printf("Pi (2 decimals): %.2lf\n", pi);
    printf("Grade: %c\n", grade);

    return 0;
}
