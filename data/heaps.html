‚úÖ What is a Heap?

A heap is a complete binary tree that satisfies the heap property:

  • Max-Heap: Every parent node is greater than or equal to its children.
  • Min-Heap: Every parent node is less than or equal to its children.

Complete binary tree means all levels are fully filled except possibly the last, which is filled from left to right.

Heaps are commonly used to implement:

  • Priority queues
  • Heap sort
  • Graph algorithms (Dijkstra, Prim)

üìå Array Representation

For a node at index i:

  • Left child ‚Üí 2*i + 1
  • Right child ‚Üí 2*i + 2
  • Parent ‚Üí (i - 1) / 2

This allows a heap to be efficiently implemented using an array without pointers.

üßæ Example 1 ‚Äì Max-Heap Using std::priority_queue 

#include <iostream>
#include <queue>
using namespace std;

int main() {
    priority_queue<int> maxHeap;

    maxHeap.push(10);
    maxHeap.push(5);
    maxHeap.push(20);
    maxHeap.push(15);

    cout << "Max-Heap elements: ";
    while (!maxHeap.empty()) {
        cout << maxHeap.top() << " ";
        maxHeap.pop();
    }
    cout << endl;
    return 0;
}

Output: 20 15 10 5

üß© Example 3 ‚Äì Manual Heapify (Array Representation) 

#include <iostream>
#include <vector>
using namespace std;

void heapify(vector<int>& arr, int n, int i) {
    int largest = i;
    int left = 2*i + 1;
    int right = 2*i + 2;

    if (left < n && arr[left] > arr[largest])
        largest = left;
    if (right < n && arr[right] > arr[largest])
        largest = right;
    if (largest != i) {
        swap(arr[i], arr[largest]);
        heapify(arr, n, largest);
    }
}

void buildHeap(vector<int>& arr) {
    int n = arr.size();
    for (int i = n/2 - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }
}

int main() {
    vector<int> arr = {3, 5, 1, 10, 2, 7};
    buildHeap(arr);

    cout << "Heap array: ";
    for (int x : arr) cout << x << " ";
    cout << endl;
    return 0;
}

Explanation:

  • Start heapifying from last non-leaf node down to root.
  • Ensures heap property is satisfied for the entire tree.

‚ö†Ô∏è Important Rules

  1. Heap property must be maintained after insertion and deletion.
  2. Insertion ‚Üí add at end, then bubble up to restore heap property.
  3. Deletion (root) ‚Üí replace root with last element, then heapify down.
  4. Array representation is efficient and avoids explicit pointers.
  5. Time complexity:
    • insert ‚Üí O(log n)
    • delete root ‚Üí O(log n)
    • buildHeap ‚Üí O(n)
  6. Heaps are widely used in priority queues and sorting algorithms (heap sort).
