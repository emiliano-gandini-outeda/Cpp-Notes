In C++, strings can be represented using character arrays, which are sequences of characters stored in contiguous memory and terminated by a special null character '\0'. This null terminator tells the program where the string ends.

üß† Declaration:

char name[6] = {'A', 'l', 'i', 'c', 'e', '\0'}; // Manual null termination
char word[] = "Hello";                          // Auto null-terminated

When using double quotes, the compiler automatically adds the '\0' at the end. Always ensure there is enough space for it when specifying array size.

üì• Input and Output:

char name[50];
scanf("%s", name);          // Reads a single word (stops at spaces)
printf("Name: %s\n", name); // Prints the string

üîÅ Accessing & Modifying Characters:

üìå Common String Functions (<string.h>):

  • strcpy(dest, src) ‚Äì Copy string
  • strcmp(s1, s2) ‚Äì Compare two strings (returns 0 if equal)
