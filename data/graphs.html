‚úÖ What is a Graph?

A graph is a data structure consisting of:

  • A set of nodes (vertices)
  • A set of edges connecting pairs of nodes

Graphs are used to model complex relationships between elements, such as:

  • Social networks
  • Transportation routes
  • Task dependencies
  • Internet connections

üß† Key Concepts

Concept Description
Vertex (Node) The basic unit of a graph
Edge A connection between two vertices
Directed Graph Edges have a direction (from one node to another)
Undirected Graph Edges have no direction
Edge Weight A numerical value representing cost, distance, or capacity

üìå Graph Representations

  1. Adjacency List
    • Each node stores a list of its connected nodes.
    • Efficient for sparse graphs.
  2. Adjacency Matrix
    • A n √ó n matrix where each cell indicates presence or weight of an edge.
    • Efficient for dense graphs.

üßæ Example ‚Äì Directed Graph Using Adjacency List 

#include <iostream>
#include <vector>
using namespace std;

int main() {
    int vertices = 5;
    vector<vector<int>> graph(vertices);

    // Add edges
    graph[0].push_back(1);
    graph[0].push_back(2);
    graph[1].push_back(2);
    graph[2].push_back(3);
    graph[3].push_back(4);

    // Print graph
    for (int i = 0; i < vertices; i++) {
        cout << "Node " << i << " -> ";
        for (int j : graph[i]) cout << j << " ";
        cout << endl;
    }

    return 0;
}

Explanation:

  • Each index of the vector represents a node.
  • The inner vectors store the adjacent nodes.
  • This structure allows traversal using algorithms like DFS and BFS.

üîÑ Traversal Example (DFS) 

#include <iostream>
#include <vector>
using namespace std;

void DFS(int node, vector<vector<int>>& graph, vector<bool>& visited) {
    visited[node] = true;
    cout << node << " ";

    for (int adj : graph[node]) {
        if (!visited[adj]) DFS(adj, graph, visited);
    }
}

int main() {
    int vertices = 5;
    vector<vector<int>> graph(vertices);
    graph[0] = {1, 2};
    graph[1] = {2};
    graph[2] = {3};
    graph[3] = {4};

    vector<bool> visited(vertices, false);

    cout << "DFS starting from node 0: ";
    DFS(0, graph, visited);
    cout << endl;

    return 0;
}

‚ö†Ô∏è Important Notes

  1. Directed vs. undirected affects traversal and connectivity.
  2. Weighted edges allow modeling costs, distances, or capacities.
  3. Choice of representation affects algorithm efficiency:
    • Adjacency list ‚Üí faster for sparse graphs
    • Adjacency matrix ‚Üí faster for dense graphs or quick edge checks
  4. Graph algorithms include: DFS, BFS, Dijkstra, Bellman-Ford, Prim, Kruskal, etc.
