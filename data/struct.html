‚úÖ What is a struct?

A struct is a user-defined data type that allows you to group variables of different types under a single name. It is often used to represent objects with multiple attributes.

üß† Syntax

struct StructName {
    type member1;
    type member2;
    // ... more members
};
  • StructName: The name of the structure.
  • member: Variables that belong to the struct.

üìå Declaring Struct Variables

StructName variable1, variable2;
  • You can declare multiple variables of the struct type at once.
  • Members are accessed using the dot operator ..

üßæ Example Program

#include <iostream>
using namespace std;

struct Person {
    string name;
    int age;
    float height;
};

int main() {
    Person p1;
    p1.name = "Emiliano";
    p1.age = 30;
    p1.height = 1.75;

    cout << "Name: " << p1.name << endl;
    cout << "Age: " << p1.age << endl;
    cout << "Height: " << p1.height << " meters" << endl;

    return 0;
}

üîÑ Array of Structs

#include <iostream>
using namespace std;

struct Point {
    int x;
    int y;
};

int main() {
    Point points[3] = {{1, 2}, {3, 4}, {5, 6}};

    for (int i = 0; i < 3; i++) {
        cout << "Point " << i << ": (" << points[i].x << ", " << points[i].y << ")\n";
    }

    return 0;
}

‚ö†Ô∏è Important Rules

  1. Struct members can be of different types.
  2. Access members with . for variables, or -> for pointers.
  3. Structs can have functions (methods) in C++.
  4. Structs can be nested inside other structs.
