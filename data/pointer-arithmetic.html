‚úÖ What is Pointer Arithmetic?

Pointer arithmetic lets you move a pointer to different memory locations by adding or subtracting integers. The movement depends on the size of the data type the pointer points to.

üß† Basic Operations

Operation Meaning
ptr + n Move forward n elements
ptr - n Move backward n elements
ptr1 - ptr2 Get the number of elements between two pointers (must point to the same array)
++ptr Move to next element
--ptr Move to previous element

üìå How It Works

  • If ptr is an int* and sizeof(int) = 4, then ptr + 1 moves 4 bytes forward.
  • Pointer arithmetic moves in units of the pointed type, not raw bytes.

üßæ Example Program 

#include <stdio.h>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *ptr = arr; // Points to first element

    printf("First element: %d\n", *ptr);
    ptr++; // Move to next element
    printf("Second element: %d\n", *ptr);

    ptr += 2; // Move forward 2 elements
    printf("Fourth element: %d\n", *ptr);

    ptr--; // Move back one element
    printf("Third element: %d\n", *ptr);

    return 0;
}

üîÑ Pointer Difference Example 

#include <stdio.h>

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int *p1 = &arr[0];
    int *p2 = &arr[4];

    printf("Elements between p1 and p2: %ld\n", p2 - p1);

    return 0;
}

‚ö†Ô∏è Important Rules

  1. Pointer arithmetic is valid only within the same array (or one past the end).
  2. Moving outside an array‚Äôs bounds causes undefined behavior.
  3. The increment/decrement size depends on sizeof(type).
  4. Works with all pointer types except void* (requires casting).
