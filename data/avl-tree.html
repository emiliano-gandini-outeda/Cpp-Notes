‚úÖ What is an AVL Tree?

An AVL tree is a self-balancing Binary Search Tree (BST) where the height difference (balance factor) between the left and right subtrees of any node is at most 1.

  • This ensures the tree remains balanced, providing O(log n) time complexity for search, insertion, and deletion.
  • Named after its inventors: Adelson-Velsky and Landis.

Key Terms:

  • Balance Factor: height(left subtree) - height(right subtree)
  • Rotation: Operation used to rebalance the tree. Types:
    • Right Rotation (LL)
    • Left Rotation (RR)
    • Left-Right Rotation (LR)
    • Right-Left Rotation (RL)

struct Node {
    int data;
    Node* left;
    Node* right;
    int height;  // Height of the node
};

üßæ Example Program ‚Äì Insertion in AVL Tree 

#include <iostream>
using namespace std;

struct Node {
    int data;
    Node* left;
    Node* right;
    int height;
};

// Get height of a node
int height(Node* node) { return node ? node->height : 0; }

// Get balance factor
int getBalance(Node* node) {
    return node ? height(node->left) - height(node->right) : 0;
}

// Right rotation
Node* rightRotate(Node* y) {
    Node* x = y->left;
    Node* T2 = x->right;

    x->right = y;
    y->left = T2;

    y->height = max(height(y->left), height(y->right)) + 1;
    x->height = max(height(x->left), height(x->right)) + 1;

    return x;
}

// Left rotation
Node* leftRotate(Node* x) {
    Node* y = x->right;
    Node* T2 = y->left;

    y->left = x;
    x->right = T2;

    x->height = max(height(x->left), height(x->right)) + 1;
    y->height = max(height(y->left), height(y->right)) + 1;

    return y;
}

// Create a new node
Node* createNode(int value) { return new Node{value, nullptr, nullptr, 1}; }

// Insert a node
Node* insert(Node* root, int value) {
    if (!root) return createNode(value);

    if (value < root->data)
        root->left = insert(root->left, value);
    else if (value > root->data)
        root->right = insert(root->right, value);
    else
        return root;  // No duplicates allowed

    root->height = 1 + max(height(root->left), height(root->right));
    int balance = getBalance(root);

    if (balance > 1 && value < root->left->data) return rightRotate(root); // LL
    if (balance < -1 && value > root->right->data) return leftRotate(root); // RR
    if (balance > 1 && value > root->left->data) { // LR
        root->left = leftRotate(root->left);
        return rightRotate(root);
    }
    if (balance < -1 && value < root->right->data) { // RL
        root->right = rightRotate(root->right);
        return leftRotate(root);
    }

    return root;
}

// In-order traversal
void inOrder(Node* root) {
    if (root) {
        inOrder(root->left);
        cout << root->data << " ";
        inOrder(root->right);
    }
}

int main() {
    Node* root = nullptr;
    root = insert(root, 10);
    root = insert(root, 20);
    root = insert(root, 30);
    root = insert(root, 40);
    root = insert(root, 50);
    root = insert(root, 25);

    cout << "In-order traversal of AVL tree: ";
    inOrder(root);
    cout << endl;

    return 0;
}

‚ö†Ô∏è Important Notes

  1. AVL trees maintain balance automatically after insertions and deletions.
  2. Rotations are used to restore balance when the balance factor is ¬±2.
  3. Guarantees O(log n) operations even in the worst case.
  4. Slightly more complex than plain BST due to height and rotation management.
