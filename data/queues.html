‚úÖ What is a Queue?

A queue is a linear data structure that follows FIFO (First In, First Out) principle. The first element inserted is the first one to be removed.

üß† Basic Operations

Operation Description
enqueue() Add an element to the rear of the queue
dequeue() Remove an element from the front of the queue
front() View the front element without removing it
rear() / back() View the last element
empty() Check if the queue is empty
size() Returns the number of elements in the queue

üìå Implementation Methods

  1. Array-based Queue ‚Äì Fixed size array with circular behavior recommended.
  2. Linked-list Queue ‚Äì Dynamic size using nodes.
  3. STL Queue ‚Äì C++ Standard Template Library implementation using queue container.

üßæ Example Program (STL Queue)

#include <iostream>
#include <queue>
using namespace std;

int main() {
    queue<int> q;
    q.push(10);
    q.push(20);
    q.push(30);

    cout << "Front element: " << q.front() << endl; // 10
    cout << "Rear element: " << q.back() << endl;   // 30

    q.pop(); // Remove 10
    cout << "Front element after dequeue: " << q.front() << endl; // 20
    cout << "Queue size: " << q.size() << endl;
    cout << "Is queue empty? " << (q.empty() ? "Yes" : "No") << endl;

    return 0;
}

üîÑ Example Using Array (Circular Queue)

#include <iostream>
#define MAX 5
using namespace std;

int queue[MAX];
int front = -1, rear = -1;

void enqueue(int x) {
    if((rear + 1) % MAX == front) cout << "Queue Overflow\n";
    else {
        if(front == -1) front = 0;
        rear = (rear + 1) % MAX;
        queue[rear] = x;
    }
}

void dequeue() {
    if(front == -1) cout << "Queue Underflow\n";
    else if(front == rear) front = rear = -1;
    else front = (front + 1) % MAX;
}

int getFront() {
    if(front == -1) cout << "Queue Empty\n";
    else return queue[front];
}

int main() {
    enqueue(10);
    enqueue(20);
    enqueue(30);

    cout << "Front element: " << getFront() << endl;
    dequeue();
    cout << "Front element after dequeue: " << getFront() << endl;

    return 0;
}

‚ö†Ô∏è Important Rules

  1. Overflow occurs if you enqueue into a full array-based queue.
  2. Underflow occurs if you dequeue from an empty queue.
  3. Circular queues avoid wasted space in arrays.
  4. STL queues are dynamic, so no overflow concerns.
