‚úÖ What are Assignment Operators?

Assignment operators are used to store a value into a variable or update its value. C++ also supports increment (++) and decrement (--) operators to increase or decrease a variable by 1.

üß† Basic Syntax

variable operator value;
variable++;    // Increment by 1
variable--;    // Decrement by 1
++variable;    // Pre-increment
--variable;    // Pre-decrement

üìå List of Operators

Operator Meaning Example Equivalent To
= Assign x = 5; x = 5;
+= Add and assign x += 3; x = x + 3;
-= Subtract and assign x -= 2; x = x - 2;
*= Multiply and assign x *= 4; x = x * 4;
/= Divide and assign x /= 2; x = x / 2;
%= Modulus and assign x %= 3; x = x % 3;
++ Increment by 1 x++; or ++x; x = x + 1;
-- Decrement by 1 x--; or --x; x = x - 1;

üßæ Example Program

#include <iostream>
using namespace std;

int main() {
    int x = 10;

    x += 5;  // x = 15
    cout << "x = " << x << endl;

    x -= 3;  // x = 12
    cout << "x = " << x << endl;

    x *= 2;  // x = 24
    cout << "x = " << x << endl;

    x /= 4;  // x = 6
    cout << "x = " << x << endl;

    x %= 5;  // x = 1
    cout << "x = " << x << endl;

    x++;     // x = 2
    cout << "x after x++ = " << x << endl;

    ++x;     // x = 3
    cout << "x after ++x = " << x << endl;

    x--;     // x = 2
    cout << "x after x-- = " << x << endl;

    --x;     // x = 1
    cout << "x after --x = " << x << endl;

    return 0;
}

‚ö†Ô∏è Important Rules

  1. ++ and -- can be pre (++x) or post (x++), with slightly different evaluation order.
  2. Combined operators (+=, -= etc.) work with all numeric types.
  3. Division by zero is still undefined behavior.
  4. ++/-- increment or decrement by 1, and can be used in loops, expressions, or standalone.
