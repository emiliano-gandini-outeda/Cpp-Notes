Arithmetic operators are symbols used to perform mathematical operations on numeric variables and values in C.

üìå Integer vs Floating-Point Division

  • Integer division: Both operands are integers ‚Üí result is an integer. Example: 7 / 2 ‚Üí 3
  • Floating-point division: At least one operand is a float ‚Üí result is decimal. Example: 7 / 2.0 ‚Üí 3.5

üßæ Example Program 

#include <stdio.h>

int main() {
    int a = 10, b = 3;

    printf("a + b = %d\n", a + b);
    printf("a - b = %d\n", a - b);
    printf("a * b = %d\n", a * b);
    printf("a / b = %d\n", a / b);   // Integer division
    printf("a %% b = %d\n", a % b);  // Modulus

    return 0;
}

üîÑ Floating-Point Example 

#include <stdio.h>

int main() {
    double x = 10.0, y = 3.0;

    printf("x / y = %.2f\n", x / y);

    return 0;
}

‚ö†Ô∏è Important Rules

  1. % can only be used with integers.
  2. Division by zero causes undefined behavior.
  3. For decimal results, at least one operand must be float/double.
  4. Operator precedence: *, /, % have higher priority than +, -.
