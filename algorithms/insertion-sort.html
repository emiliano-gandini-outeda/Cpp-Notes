‚úÖ What is Insertion Sort?

Insertion Sort is a simple comparison-based sorting algorithm.

  • It builds the sorted array one element at a time.
  • At each step, it picks the next element from the unsorted portion and inserts it into the correct position in the sorted portion.
  • Efficient for small arrays or nearly sorted arrays.

üìå How Insertion Sort Works

  1. Start with the second element (assume first element is already sorted).
  2. Compare it with elements in the sorted portion from right to left.
  3. Shift elements that are greater to the right.
  4. Insert the current element in its correct position.
  5. Repeat for all remaining elements.

üßæ Example ‚Äì Standard Insertion Sort 

#include <iostream>
using namespace std;

void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}

int main() {
    int arr[] = {12, 11, 13, 5, 6};
    int n = sizeof(arr)/sizeof(arr[0]);
    insertionSort(arr, n);
    cout << "Sorted array: ";
    for (int i = 0; i < n; i++) cout << arr[i] << " ";
    cout << endl;
    return 0;
}

Output: Sorted array: 5 6 11 12 13

üîÑ Step-by-Step Explanation

  • Iteration 1: Key = 11 ‚Üí Compare with 12 ‚Üí Shift 12 ‚Üí Insert 11
  • Iteration 2: Key = 13 ‚Üí Already in correct position ‚Üí No shifts
  • Iteration 3: Key = 5 ‚Üí Shift 12, 11 ‚Üí Insert 5
  • Iteration 4: Key = 6 ‚Üí Shift 12 ‚Üí Insert 6
  • The sorted portion grows from left to right, while the unsorted portion shrinks.

‚ö†Ô∏è Important Rules

  1. Stable algorithm ‚Üí preserves relative order of equal elements.
  2. In-place sorting ‚Üí requires O(1) extra memory.

  3. Time complexity:

    • Worst-case: O(n¬≤) ‚Üí array sorted in reverse order
    • Best-case: O(n) ‚Üí array already sorted
    • Average-case: O(n¬≤)
  4. Efficient for small or nearly sorted arrays, but inefficient for large arrays.
  5. Works well as a subroutine in more advanced algorithms like Timsort.
