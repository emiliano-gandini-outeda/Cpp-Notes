‚úÖ What is Linear Search?

Linear Search (or Sequential Search) is the simplest searching algorithm.

  • It checks each element of an array (or list) one by one until it finds the target element.
  • If the element is found, it returns its position.
  • If the element is not found, it returns -1 or indicates ‚Äúnot found.‚Äù
  • Works on unsorted or sorted arrays.

üìå How Linear Search Works

  1. Start from the first element of the array.
  2. Compare it with the target element.
  3. If it matches, return the index.
  4. If it doesn‚Äôt, move to the next element.
  5. Repeat until the end of the array.
  6. If the element is not found, return -1.

üßæ Example ‚Äì Linear Search 

#include <iostream>
using namespace std;

int linearSearch(int arr[], int n, int target) {
    for (int i = 0; i < n; i++) {
        if (arr[i] == target)
            return i; // Element found
    }
    return -1; // Element not found
}

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int n = sizeof(arr) / sizeof(arr[0]);
    int target = 30;

    int index = linearSearch(arr, n, target);

    if (index != -1)
        cout << "Element found at index: " << index << endl;
    else
        cout << "Element not found" << endl;

    return 0;
}

Output:

Element found at index: 2

‚ö†Ô∏è Important Rules

  1. Works on both sorted and unsorted arrays.

  2. Time complexity:

    • Best-case: O(1) ‚Üí element is at the first position
    • Worst-case: O(n) ‚Üí element is at the last position or not present
    • Average-case: O(n) ‚Üí element is somewhere in the middle
  3. Space complexity: O(1) ‚Üí no extra memory needed
  4. Simple to implement but inefficient for large arrays compared to Binary Search.
