‚úÖ What is Breadth-First Search?

Breadth-First Search (BFS) is a graph traversal algorithm that explores the graph level by level, visiting all nodes at the current depth before moving to the next.

  • Starts at a source node.
  • Explores all neighboring nodes first, then moves to their neighbors.
  • Implemented using a queue to maintain the order of nodes.
  • Works for both directed and undirected graphs.

üß† Key Concepts

Concept Description
Traversal type Level-order, breadth-first
Graph type Directed or undirected
Implementation Queue-based
Visited tracking Keeps track of visited nodes to avoid revisiting
Time complexity O(V + E) ‚Üí V = vertices, E = edges
Space complexity O(V) for the queue
Use cases Shortest path in unweighted graphs, connectivity, bipartite check

üìå How BFS Works

  1. Start at the source node.
  2. Mark it as visited and enqueue it.

  3. While the queue is not empty:

    • Dequeue a node u.
    • Visit u (e.g., print or process).

    • For each unvisited neighbor of u:

      • Mark as visited
      • Enqueue neighbor
  4. Repeat until all reachable nodes are visited.

üßæ Example ‚Äì BFS using adjacency list

#include <iostream>
#include <vector>
#include <queue>
using namespace std;

void bfs(int start, vector>& graph, vector& visited) {
    queue q;
    visited[start] = true;
    q.push(start);

    while (!q.empty()) {
        int node = q.front();
        q.pop();
        cout << node << " ";

        for (int neighbor : graph[node]) {
            if (!visited[neighbor]) {
                visited[neighbor] = true;
                q.push(neighbor);
            }
        }
    }
}

int main() {
    int vertices = 5;
    vector> graph(vertices);

    // Add edges
    graph[0] = {1, 2};
    graph[1] = {2};
    graph[2] = {0, 3};
    graph[3] = {3, 4};

    vector visited(vertices, false);

    cout << "BFS traversal starting from node 0: ";
    bfs(0, graph, visited);
    cout << endl;

    return 0;
}

üîÑ Step-by-Step Example

Graph (edges):

0 ‚Üí 1, 0 ‚Üí 2, 1 ‚Üí 2, 2 ‚Üí 0, 2 ‚Üí 3, 3 ‚Üí 3, 3 ‚Üí 4

BFS starting at node 0:

  1. Start with queue = [0], visit 0 ‚Üí neighbors 1,2 ‚Üí enqueue ‚Üí queue = [1,2]
  2. Dequeue 1 ‚Üí visit ‚Üí neighbor 2 already visited ‚Üí queue = [2]
  3. Dequeue 2 ‚Üí visit ‚Üí neighbors 0,3 ‚Üí 0 visited, enqueue 3 ‚Üí queue = [3]
  4. Dequeue 3 ‚Üí visit ‚Üí neighbors 3,4 ‚Üí 3 visited, enqueue 4 ‚Üí queue = [4]
  5. Dequeue 4 ‚Üí visit ‚Üí no unvisited neighbors ‚Üí queue empty ‚Üí finish

Traversal order: 0 1 2 3 4 ‚úÖ

‚ö†Ô∏è Important Rules

  1. BFS finds the shortest path in unweighted graphs.
  2. Requires visited tracking to avoid infinite loops in cyclic graphs.
  3. Time complexity: O(V + E) ‚Üí visits each vertex and edge once.
  4. Space complexity: O(V) ‚Üí queue stores nodes at each level.

  5. BFS is often used for:

    • Finding shortest paths
    • Checking connectivity
    • Bipartite graph check
    • Level-order traversal in trees
